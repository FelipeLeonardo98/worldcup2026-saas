# World Scout

A static MVP SaaS prototype for searching World Cup 2026 players by
national-team shirt number or name, then returning their current club, club
number, height, weight, age, and position.

Open `index.html` in a browser to use it.

## MVP scope

- Brazil, France, Germany, and Argentina.
- Local seeded dataset in `script.js`.
- Exact numeric search for shirt numbers.
- Flexible name, alias, club, and position search.
- Position filter for quick roster browsing.
- Team selector with per-team quick searches and visual theme.

## Source notes

The initial source list came from `info.md`, with API-Football, TheSportsDB,
Transfermarkt, Sofascore, Sportradar, and Ogol suggested as future integrations.
For this static MVP, squad numbers were seeded from recent 2026 squad references
and physical profile data was captured as a product-prototype snapshot.
